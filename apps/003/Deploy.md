# デプロイ手順 (GitHub Pages)

このアプリケーションをGitHub Pagesで公開する手順は以下の通りです。

## 1. リポジトリの準備
1. GitHubで新しいリポジトリを作成します。
2. 作成した以下のファイルをリポジトリにアップロード（push）します。
   - `index.html`
   - `styles.css`
   - `app.js`

## 2. GitHub Pagesの設定
1. GitHubリポジトリの画面上部にある **Settings** タブをクリックします。
2. 左サイドメニューの **Pages** を選択します。
3. **Build and deployment** セクションの **Branch** で `main` (または `master`) ブランチを選択し、フォルダを `/ (root)` に設定して **Save** をクリックします。

## 3. 公開確認
1. 数分待つと、画面上部に `Your site is live at ...` というURLが表示されます。
2. そのURLにアクセスして、アプリが動作することを確認してください。

## 注意事項
- **API制限:** 本アプリで使用している Nominatim (ジオコーディング) は、短時間に大量のリクエストを送ると一時的にブロックされる可能性があるため、連続したテストの際はご注意ください。
- **HTTPS:** GitHub PagesはデフォルトでHTTPSで提供されます。位置情報取得（GPS）機能はセキュリティ上、HTTPS環境下でのみ動作するため、ローカルファイル（file://）として開くのではなく、サーバー経由（GitHub Pages等）で利用してください。
